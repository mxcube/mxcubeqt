Module description
--------------------------
Centring math module is used to calculate motor positions for centring proceedure.

**Coordinate system**

Camera axes directions:

.. math::
   \vec r^{x}_{cam} = {1,0,0}

   \vec r^{y}_{cam} = {0,1,0}
Beam position in camera coordinates:

.. math::
   x_{beam}, y_{beam}
